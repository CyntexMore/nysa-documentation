## Features

<CardGrid stagger>
	<Card title="Chat with Nysa" icon="discord">
	    Chat with Nysa on Discord, Telegram, and Slack. Not just text, but voice calls are also supported on Discord. Extendible via the SDK!
	</Card>
	<Card title="Play Games with Nysa" icon="rocket">
	    You can play Minecraft with Nysa out of the box, or you can implement any other game via the SDK.
	</Card>
	<Card title="BYO API" icon="pencil">
	    Self-host your own instance of Nysa and bring your own API and keys. If any API you'd like to use is not implemented, you can easily do it yourself with the SDK.
	</Card>
	<Card title="Customize Nysa" icon="setting">
	    You can easily customize Nysa's persona and LLM settings through a `.env` file or the web interface.
	</Card>
	<Card title="Search Plugins" icon="open-book">
	    Easily search for Nysa plugins and platform/game integrations in the marketplace.
	</Card>
