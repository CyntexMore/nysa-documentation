## 機能

<CardGrid stagger>
	<Card title="Nysaとチャット" icon="discord">
	    Discord、Telegram、SlackでNysaとチャットできます。テキストだけでなく、Discordでは音声通話もサポートされています。SDKを通じて拡張可能！
	</Card>
	<Card title="Nysaとゲームをプレイ" icon="rocket">
	    Minecraftはすぐにプレイできますが、SDKを通じて他のゲームも実装できます。
	</Card>
	<Card title="BYO API" icon="pencil">
	    Nysaのインスタンスをセルフホストし、独自のAPIとキーを持ち込みましょう。使用したいAPIが実装されていなくても、SDKで簡単に自分で実装できます。
	</Card>
	<Card title="Nysaをカスタマイズ" icon="setting">
	    `.env`ファイルまたはWebインターフェースを通じて、NysaのペルソナとLLM設定を簡単にカスタマイズできます。
	</Card>
	<Card title="プラグインを検索" icon="open-book">
	    マーケットプレイスでNysaのプラグインやプラットフォーム/ゲーム統合を簡単に検索できます。
	</Card>
</CardGrid>
